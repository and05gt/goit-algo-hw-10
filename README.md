# goit-algo-hw-10

## Linear programming and randomized algorithms

## Порівняльний аналіз методів обчислення визначеного інтеграла

Метою роботи було обчислення значення визначеного інтеграла функції $f(x) = x^2$
на інтервалі $[0, 2]$.

Обчислення проводилося двома методами:

1. **Метод Монте-Карло** (алгоритм Hit-or-Miss).
2. **Функція `quad`** з бібліотеки SciPy (використовується як еталонне значення,
   оскільки базується на високоточних чисельних методах, таких як квадратура
   Гаусса-Кронрода).

### Вхідні дані

-   **Функція:** $f(x) = x^2$
-   **Межі інтегрування:** $a = 0, b = 2$
-   **Кількість експериментів (точок) для Монте-Карло:** $N = 1,000,000$

---

### Результати розрахунків

#### 1. Аналітичне (точне) значення

Математичне обчислення інтеграла:
$$\int_0^2 x^2 \,dx = \left[ \frac{x^3}{3} \right]_0^2 = \frac{2^3}{3} - \frac{0^3}{3} = \frac{8}{3} \approx 2.666666...$$

#### 2. Результат функції `scipy.integrate.quad`

-   **Значення:** `2.6666666666666665`
-   **Абсолютна похибка (estimate):** `2.96e-14` (майже нульова)

#### 3. Результат методу Монте-Карло

_(Примітка: через випадковий характер методу значення може незначно відрізнятися
при кожному запуску)_

-   **Отримане значення:** `~2.668` (середнє значення при $10^6$ точок)
-   **Відхилення від точного:** `~0.001 - 0.003`

---

### Висновки

1. **Правильність розрахунків:** Код реалізує метод Монте-Карло геометричним
   способом правильно. Висота прямокутника була коректно задана як $f(b) = 4$,
   що дозволило охопити весь діапазон значень функції на відрізку. Результат,
   отриманий алгоритмом, дуже близький до точного значення.

2. **Точність методів:**

    - Функція `quad` дає результат, що збігається з аналітичним розрахунком із
      точністю до машинного епсилона (15-16 знаків після коми).
    - Метод Монте-Карло при $1,000,000$ точок дає наближене значення з похибкою
      приблизно в 3-му знаку після коми. Це підтверджує, що метод є
      імовірнісним: його точність залежить від кількості випадкових точок ($N$)
      і зростає пропорційно $\sqrt{N}$.

3. **Підсумок:** Метод Монте-Карло успішно впорався із завданням наближеного
   інтегрування. Хоча для одновимірних гладких функцій (як $x^2$) використання
   `quad` є доцільнішим через швидкість і точність, реалізований алгоритм
   Монте-Карло працює коректно і може бути застосований для складніших завдань,
   де аналітичні методи або стандартні квадратури менш ефективні.
